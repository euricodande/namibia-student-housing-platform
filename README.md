# Namibia Student Housing Platform

A full-stack web application that helps students find rental accommodation and allows landlords to manage listings, enquiries, rentals, and maintenance requests.

## Tech Stack

### Backend
- Java
- Spring Boot
- PostgreSQL
- Spring Security
- JWT Authentication
- Spring Data JPA
- Flyway

### Frontend
- Next.js
- TypeScript
- Tailwind CSS

## Main Features

- Student and landlord registration
- JWT-based login
- Public listing browsing
- Landlord listing management
- Admin approval of landlords and listings
- Saved listings
- Enquiries
- Rentals
- Maintenance requests
- Admin dashboard
